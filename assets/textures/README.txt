Place planet textures here using these exact filenames:

- sun.jpg
- mercury.jpg
- venus.jpg
- earth.jpg
- mars.jpg
- jupiter.jpg
- saturn.jpg
- uranus.jpg
- neptune.jpg

Recommended format: equirectangular maps (2:1 aspect ratio), JPG or PNG.
Recommended resolution: 1024x512 up to 4096x2048.

If a texture file is missing, the app automatically falls back to a color-rendered planet.
